# 网站修改指南

## 1. 删除导航栏标签

### 问题
网页导航栏中的标签太多（如 Talks、Teaching、Portfolio、Blog、CV、Guide），需要删除某些标签。

### 解决方法
编辑 `_data/navigation.yml` 文件，删除不需要的导航项。

**文件位置：** `_data/navigation.yml`

**示例：删除 Talks 和 Teaching**

找到以下内容：
```yaml
main:
  - title: "Publications"
    url: /publications/

  - title: "Talks"
    url: /talks/

  - title: "Teaching"
    url: /teaching/

  - title: "Portfolio"
    url: /portfolio/
```

删除 Talks 和 Teaching 两项后：
```yaml
main:
  - title: "Publications"
    url: /publications/

  - title: "Portfolio"
    url: /portfolio/
```

**注意：** 修改后需要重启 Jekyll 服务才能看到效果。

## 2. 添加灰色分割线

### 问题
在页面的某些位置需要添加灰色的水平分割线。

### 解决方法
在 Markdown 文件中使用以下任一方式添加分割线。

**适用文件：** `_pages/*.md`、`_posts/*.md` 等所有 Markdown 文件

### 方法一：三个连字符
```markdown
---
```

### 方法二：三个星号
```markdown
***
```

### 方法三：三个下划线
```markdown
___
```

### 方法四：HTML 标签（推荐，可自定义样式）
```html
<hr>
```

**自定义样式：**
```html
<hr style="border: 1px solid #ccc;">
<hr style="border: 2px solid #999; margin: 20px 0;">
```

### 使用示例
```markdown
## 教育背景

- 2020-2024 北京大学 博士

---

## 研究方向

- 心血管精准医学
- 生物信息学
```

**效果：** 在"教育背景"和"研究方向"之间会显示一条水平分割线。
